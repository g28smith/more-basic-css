Learning CSS

This page demonstrates how CSS works. Right now it looks plain because we haven't added any styles yet.

What is CSS?

CSS stands for Cascading Style Sheets. It controls how HTML elements look on the page.

Without CSS, every webpage would look like this: plain black text on a white background.

Why Use CSS?

CSS separates content from presentation. This means you can change the appearance of your entire website by editing just one file.

Basic CSS Concepts

  • Selectors choose which elements to style
  • Properties define what aspect to change
  • Values specify how to change it
